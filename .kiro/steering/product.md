# Product Overview

## What This Is

The **AurigaIT Python AI-SDLC Framework** — a shared set of skills, rules, and standards that AI coding agents use to generate consistent, production-quality Python code across all AurigaIT projects.

## Purpose

- Encode 18+ years of Python best practice into structured skill files
- Ensure every AI agent (regardless of tool) produces code that meets enterprise standards
- Provide a single source of truth for coding conventions, architecture patterns, and operational practices

## Target Users

- Python developers using AI coding assistants
- Tech leads enforcing standards across teams
- New developers onboarding to AurigaIT projects

## Key Features

- 22+ skill files covering code style through production operations
- 4 framework-specific skills (FastAPI, Django, DRF, Flask)
- QA agent definition for automated quality pipelines
- Reference implementation in `examples/fastapi-app/`

## Business Objectives

- Reduce code review cycles by 50% (agent follows standards pre-review)
- Onboard new developers in < 1 week (agent teaches standards in context)
- Zero security incidents from hardcoded secrets or missing auth
- Consistent architecture across all Python services
